INSTRUKSI PENEMPATAN LOGO
==========================

Letakkan file logo dengan nama: LOGO_INAKU.png

File ini akan digunakan di header aplikasi.

Format yang didukung:
- PNG (recommended untuk transparansi)
- JPG/JPEG
- WebP

Ukuran yang direkomendasikan:
- Height: 40-60px
- Width: proportional (biasanya 100-200px)
- Resolusi: @2x dan @3x untuk berbagai device density

Jika logo belum ada, aplikasi akan menampilkan teks "INAKU" sebagai fallback.
